### StakePi

Monitoring staking status with a Raspberry Pi

#### Hardware

Can easily be varied.

- [Raspberry Pi 3 B+]()
- [Qwiic Hat]()
- [BME280]()
- [Qwiic LED Stick]()
- [Micro OLED]()

#### Setup

```
pip2 install -r requirements.txt
```

And install [this fork](https://github.com/cwgoes/Adafruit_Python_SSD1306) of the SSD1306 library.

#### Usage

```bash
./monitor.py
```
